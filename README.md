# Flashcards App (Project 3 - Part 2)

## Overview

This is a flashcards web application built with React and Vite.

Users can study computer science concepts by flipping cards, submitting answers, tracking streaks, and navigating through the flashcard set.

---

## Features

### Required Features

* [x] Displays title of card set
* [x] Displays short description
* [x] Displays total number of cards
* [x] Stores card data as question/answer pairs
* [x] Shows one card at a time
* [x] Clicking card flips between question and answer
* [x] Clicking card again flips back
* [x] Input box allows users to enter a guess
* [x] Submit button checks answers
* [x] Correct answers display visual feedback
* [x] Incorrect answers display visual feedback
* [x] Next button moves to the next card in sequence
* [x] Previous button moves to the previous card in sequence
* [x] Previous button is disabled on the first card
* [x] Next button is disabled on the last card

### Stretch Features

* [x] Current streak counter
* [x] Longest streak counter
* [x] Case-insensitive answer matching
